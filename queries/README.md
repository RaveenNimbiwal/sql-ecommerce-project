This folder has all the SQL queries that were written for this project. 
The queries are divided into four files based on their usage. 

## basic_questions_1.sql and basic_questions_2.sql
These file contains basic queries 

## advance_questions.sql 
This file contains advance queries used to solve complex problems

## analytical_questions.sql
This file contains questions that focus on business problems.
The queries are used to learn more about sales, customers, and products.
